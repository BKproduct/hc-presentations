# Health Copilot — объединённая клиентская презентация

Скомпоновано из актуальной **клиентской** версии (`pro-trener-teaser`) и смыслового **моста** из логики «клиника / время приёма / разрозненные данные» (лондонский тизер без тарифов и города).

- **HTML:** `index.html` (прокрутка вниз, те же `assets/`).
- **Публично:** после push — `https://bkproduct.github.io/hc-presentations/hc-combined-deck/?v=9`

## Figma

MCP Figma в Cursor должен быть включён и авторизован (`https://mcp.figma.com/mcp` в `mcp.json`). Импорт векторной вёрстки из HTML обычно делают плагином **html.to.design** (вставить URL страницы выше) или вручную по скриншотам секций.
