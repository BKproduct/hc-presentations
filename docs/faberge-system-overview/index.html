Processing logic · 8 steps

1
Collect (parallel)
ThreadPool over N sources → RawCollection per source. Raw JSON stored in data/raw/<week>/.
2
Normalise + lexicon tag
Unified Signal schema. Tag by category (product · style · stones · cuts · metals).
3
Score
strength = vol_norm × (0.4 + 0.6·intent) × recency × source_weight.
4
Aggregate themes
(category, term) aggregate with strength + count + sources + regions + examples.
5
Z-score vs history
26-week rolling baseline. Trend labels: surge · rising · stable · fading · collapse.
6
Cohort tag
Cohort signal matching with weighted winner selection (UHNW/HENRY/Asia UHNW/Gen-Z).
7
Synthesis (Sonnet)
Cached prompt + strict JSON schema (headline · scorecard · briefs · triggers · risks).
8
Render + draft
Jinja → HTML → Playwright → PDF + draft email for final human review.
