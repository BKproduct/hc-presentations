# Скриншоты для презентации (клиентский показ)

Файлы рядом с `index.html`; в разметке используются пути `assets/…`.

| Файл | Содержимое |
|------|------------|
| `01-brief-doctor.png` | Бриф для врача |
| `02-telegram-food-fish.png` | Telegram: разбор блюда по фото + персональный контекст |
| `03-telegram-morning.png` | Telegram: утренний Health Briefing |
| `05-journal-daytime.png` | Telegram: дневной журнал (паттерны самочувствия / режима) |

**Обновление:** замените PNG теми же именами или добавьте новый файл и обновите `src` в `index.html`.
