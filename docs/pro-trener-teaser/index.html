Формулировки из генетического пайплайна v3: независимые роли, затем согласование конфликтов и отчёт в Telegram / Markdown.

Clinical GeneticistClinVar, pathogenic variants, BRCA / Lynch / HFE — только Grade A, если так задано политикой.
PharmacogenomicistCPIC Level A, CYP2C9, SLCO1B1, VKORC1 — зона дозировок и NSAID.
NutrigenomicistMTHFR, APOE, FADS1 — железо, D, B12, ответ на ω-3.
BiostatisticianQC по находкам, пробелы чипа, конфликты → флаг на WGS.

Два выхода, которые владелец студии узнает сразу.
